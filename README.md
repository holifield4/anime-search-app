# Anime Search App

A simple application to search and view anime details

## Tech Stack

- React with vite
- TypeScript
- TailwindCSS
- Redux
- React-router-dom

## Getting Started

Run the following commands:

```js
npm install
npm run dev
```

The app should be running locally in port 4000

### More
All data are provided by [Jikan REST API](https://docs.api.jikan.moe/#tag/anime)
